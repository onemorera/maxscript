Эти скрипты для 3ds Max. С 2021 работает, другие не проверялось
Они решают следующую проблему. Различные объекты max, такие как линия, сплайн, сетка, editablepoly, имеют свои собственные attach, detach, weld, breack
Скрипты позволяют использовать одну горячую клавишу для всех одинаковых действий
Они автоматически определяют тип объекта и вызывают нужную для этого типа команду

Smart_call_attach может работать с rectangl и circle. Он преобразует объект в editablepoly, а затем применяет attach
Он работает с бокс, сферой, цилиндром и конусом, но с ошибкой, объект конвертится и аттач включается, но почему-то сбрасывается

Инсталяция:
Бросаем все скрипты в окно макса, в хоткей едитор в поле экшен набираем _call, назначаем хоткеи



These scripts for 3dsMax
They solve the following problem. Different max object such as line, spline, mesh, editablepoly have its own attach, detach, weld, breack.
Scripts alow use one hotkey for all saim action
They automaticaly detected type of object and call necessary action

Smart_call_attach can work with rectangl and circle. It converts the object to editablepoly and then apply attach
It works with box, sphere, cylindr and cone but has some bug
